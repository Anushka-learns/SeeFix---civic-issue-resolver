# SeeFix---civic-issue-resolver
An AI-powered reporting system where citizens upload images and the system automatically detects the issue and notifies municipal staff.

User logs in

     ↓
     
Chooses report issue

     ↓
Uploads image

     ↓
Backend API receives request

     ↓
Image sent to AI model

     ↓
Model predicts issue type/relevency 

     ↓
Complaint stored in database

     ↓
Municipal dashboard shows issue

     ↓
Admin logs in

     ↓
Employee marks issue resolved/not
